1. Chạy Device trên Android Studio (emulator-5554).
2. Cài đặt server Appium (Nodejs).
3. Cài đặt UIAutomator2
4. Cài đặt cấu hình trên cả Katalon và Robot Framework.
```
${REMOTE_URL}       http://127.0.0.1:4723
${DEVICE_NAME}      emulator-5554
${PLATFORM_NAME}    Android
${APP_PACKAGE}      com.swaglabsmobileapp
${APP_ACTIVITY}     .MainActivity
${TIMEOUT}          10s
```

5. Chạy Script.
